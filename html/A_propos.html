À PROPOS

Vers l'âge de 27 ans, j'ai commencé deux trois sites internet perso, pricipalement en tant que autodidacte. Ensuite j'ai continué à me spécialiser dans le développement web sur la plateforme OpenClassroom.

À la suite de cela, j'ai effectué une formation de développeur web / web mobile à l'école SIMPLON. Cela m'a permis de développer mes compétences dans le domaine du web, comme le travail d'équipe sur un projet commun ou encore les divers langues de programmation.

Depuis ce début d'année 2022 je suis développeur web en freelance. Étant basé dans les Ardennes, je propose mes Services dans toute la France, en travail à distance pour accompagner les entreprises dans la création de leur projet web ou la mise à jour de celui-ci.
